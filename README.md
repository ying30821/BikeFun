# BikeFun
![image](https://github.com/ying30821/BikeFun/blob/master/src/assets/images/readmePic.png?raw=true)

作品連結：[點擊作品連結](https://ying30821.github.io/BikeFun/#/)

## 功能說明
- 串接交通部TDX API
- 瀏覽全台公共自行車租借站、自行車道位置
- 依照縣市、關鍵字篩選自行車租借站、自行車道位置

## 使用技術
- Vue CLI
- Vue Router
- vue-axios
- Bootstrap 5
- jsSHA
- Leaflet
- Leaflet MarkerCluster
